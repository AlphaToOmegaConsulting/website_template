Rejoignez-nous pour un atelier intensif sur React où vous apprendrez les techniques avancées utilisées par les développeuses professionnelles. Cet atelier pratique vous permettra de maîtriser les concepts complexes de React et d'améliorer vos compétences en développement frontend.

## Ce que vous apprendrez

- Hooks personnalisés et patterns avancés
- Optimisation des performances avec React
- Gestion d'état avec Context API et Zustand
- Tests unitaires et d'intégration
- Bonnes pratiques et architecture d'application

## Prérequis

- Connaissance de base de React
- Expérience avec JavaScript ES6+
- Ordinateur portable avec Node.js installé

L'atelier sera animé par des développeuses expérimentées de l'industrie qui partageront leurs connaissances et répondront à toutes vos questions.
